# SPI/AI Final Project Submission

This folder contains the final workflow for the Scenic Potential Index (SPI) and Accessibility Index (AI) project for the GB + North KP + AJK study area.

## Main Files

- `final_spi_ai_submission.ipynb` - final Jupyter notebook with markdown sectioning and comments.
- `data/processed/final_tehsil_spi_ai_analysis.csv` - final processed dataset used for analysis.
- `data/processed/final_tehsil_spi_ai_analysis.geojson` - spatial version of the final processed dataset.
- `data/processed/spi_index.tif` - final SPI raster.
- `data/processed/ai_index.tif` - final AI raster.
- `data/processed/spi_ai_combined.tif` - combined normalized SPI/AI raster.
- `outputs/statistical_analysis/` - figures and statistical output tables generated by the analysis.

## How to Run

1. Open `final_spi_ai_submission.ipynb` from this folder.
2. Use the Python/SDS environment used in class or any environment with the libraries listed below.
3. Run all cells from top to bottom.

The notebook resolves paths automatically from its location, so no manual directory edits are required if the folder structure is preserved.

## Required Folder Structure

The notebook expects these folders to remain in the project root:

- `data/raw/`
- `data/interim/`
- `data/processed/`
- `outputs/`

The final notebook starts from the preprocessed and grid-aligned inputs in `data/interim`, then regenerates the processed SPI/AI rasters, tehsil zonal statistics, statistical analysis outputs, and final processed dataset.

## Additional Libraries

In addition to the standard class libraries, this workflow uses:

- `geopandas` for vector spatial data.
- `rasterio` for raster reading, writing, and metadata handling.
- `numpy` and `pandas` for array and table processing.
- `matplotlib` for plots and maps.
- `libpysal` and `esda` for Queen contiguity weights, Moran's I, and LISA spatial clustering. The final notebook includes a built-in fallback if these are not installed.
- `statsmodels` for OLS regression and diagnostic tests. The final notebook includes a built-in fallback if it is not installed.
- `scipy` for distance transform and terrain/accessibility calculations.

## Final Dataset Choice

The submitted final processed dataset is `data/processed/final_tehsil_spi_ai_analysis.csv`. It is preferred over raw clipped rasters because it is the actual analysis-ready tehsil-level dataset containing SPI, AI, scenic-access gap, priority rankings, spatial statistics, regression outputs, component summaries, and sensitivity results.
