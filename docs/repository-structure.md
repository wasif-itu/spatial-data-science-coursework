# Repository Structure

This repo is organized as a coursework archive rather than a single application or project.

## Top-Level Folders

- `notebooks/core/`: main course notebooks by topic.
- `notebooks/foundations/`: introductory spatial data notebooks.
- `notebooks/geostatistics/`: variography and geostatistical exploration.
- `notebooks/practice/`: short practice notebooks and scratch work.
- `assignments/`: class assignments and activity notebooks.
- `exercises/remote-sensing/`: Landsat and raster processing exercises.
- `datasets/`: course and example datasets used by notebooks.
- `docs/`: local notes, commands, and repository documentation.

## Maintenance Notes

- Keep final project work out of this repo.
- Put new course notebooks in the most specific `notebooks/` subfolder.
- Put submitted work in `assignments/`.
- Put datasets under `datasets/`, grouped by source or exercise.
- Do not commit caches, virtual environments, notebook checkpoints, or generated figures unless they are intentionally part of the coursework record.
